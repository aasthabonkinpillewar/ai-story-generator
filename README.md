# AI-Based Multilingual Story Generator

## Project Overview
This project is an AI-based storytelling application that generates complete, meaningful stories without requiring the user to write prompts. The system takes structured inputs such as language, mood, and emotional intensity and produces a story with characters, emotional flow, and a clear conclusion.

## Selected Track
LLM / NLP-based AI Application

## Problem Statement
Many individuals struggle to express emotions through writing, especially when creating structured stories that require emotional depth, characters, and meaningful endings.

## Objective
To build an AI-based system that automatically generates well-structured and emotionally rich stories using predefined user inputs such as language, mood, and intensity.

## Features
- Multilingual story generation (English, Hindi, Marathi)
- Mood-based storytelling
- Emotional intensity control
- No prompt writing required
- Structured story flow (introduction, middle, conclusion)

## Technologies Used
- Python
- NLP techniques
- Large Language Model (LLM)
- Google Colab
- GitHub

## How to Run
1. Open `story_generator.ipynb`
2. Run all cells sequentially
3. Select language, mood, and intensity
4. View the generated story output
